# 哔哩哔哩网课批量重命名
> 哔哩哔哩uwp版本：2.14.79.0
## 代码功能
将哔哩哔哩uwp中下载的网课批量更名

> 注：本代码仅用于学习

## 实现逻辑
根据MP4目录下的.info文件命名标题

## 使用方法
根据自己软件设置的下载目录，更改第十三行的download_dir后，即可运行
